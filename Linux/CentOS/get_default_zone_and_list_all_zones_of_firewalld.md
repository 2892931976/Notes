
# Get Default Zone and List All Zones of firewalld

#### Get Default Zone

    sudo firewall-cmd --get-default-zone

#### Get Active Zones

    sudo firewall-cmd --get-active-zones

#### List All Zones and Details

    sudo firewall-cmd --list-all-zones
