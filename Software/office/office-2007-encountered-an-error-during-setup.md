# Office 2007 Encountered an Error During Setup

#### Root Cause
* Previous Office Installation Information Should be Cleared
  * `c:\ProgramData\Microsoft Help\Rgstrtn.lck`

#### How to Fix
* Delete `c:\ProgramData\Microsoft Help\` folder
