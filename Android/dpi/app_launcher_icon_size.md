
# App Launcher Icon Size

* MDPI(baseline) - 36 x 36 px
* TVDPI - 48 x 48 px
* HDPI - 72 x 72 px
* XHDPI - 96 x 96 px
* XXHDPI - 144 x 144 px
* XXXHDPI - 192 x 192 px
* Icon for Google Play Store - 512 x 512 px
