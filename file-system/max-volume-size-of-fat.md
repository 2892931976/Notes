# Max Volume Size of FAT

#### FAT16

| Description | Limit |
| :----------- | :----- |
| Max File Size | 2^32 - 1 bytes |
| Max Volume Size | 4 GB |
| Files per Volume | 2^16 |

#### FAT32

| Description | Limit |
| :----------- | :----- |
| Max File Size | 2^32 - 1 bytes |
| Max Volume Size | 32 GB |
| Files per Volume | Approximately 4 million |
